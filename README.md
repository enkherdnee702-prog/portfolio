# University Info Website

This project is a personal website that showcases information about universities, along with details about myself, including my background, skills, interests, and projects. The website is designed to be user-friendly and mobile-responsive.

## Project Structure

The project consists of the following files and directories:

```
university-info-website
├── src
│   ├── index.html         # Main HTML file for the home page
│   ├── about.html         # HTML file containing information about me
│   ├── projects.html      # HTML file showcasing my projects and goals
│   ├── assets             # Directory for images and other assets
│   │   ├── profile.jpg    # Profile picture used on the home page
│   │   └── universities    # Directory for university-related images
│   │       └── [university-images].jpg
│   ├── css                # Directory for CSS files
│   │   └── styles.css     # CSS file for styling the website
│   └── js                 # Directory for JavaScript files
│       └── main.js        # JavaScript file for interactive features
├── README.md              # Documentation for the project
```

## Features

- **Home Page**: Displays my name, a short introduction, and a profile picture, along with a navigation menu to other pages.
- **About Me Page**: Contains details about my background, skills, and interests.
- **Projects/Achievements Page**: Showcases my school projects, activities, and future goals.
- **Responsive Design**: The website is designed to be mobile-friendly, ensuring a good user experience on all devices.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open the `src/index.html` file in a web browser to view the website.
